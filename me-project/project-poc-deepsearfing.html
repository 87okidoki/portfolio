MNC PoC - deepSearfing

마인즈앤컴퍼니의 AI LLM 기반의 POC 제품 작업들 입니다.

Overview

GPT 기반 의미 검색 솔루션 딥서핑
딥서핑은 키워드 중심의 기존 검색과 달리, 수많은 데이터를 담은 다양한 문서를 서핑하듯이 넘나들며 필요한 정보를 찾아내는 GPT 기반 검색 솔루션입니다.
방대한 기업의 DB를 자연어로 더 빠르고, 더 정확하게 탐색할 수 있는 딥서핑과 함께 새롭고 청량한 검색을 경험 할 수 있습니다.

Concept

search + surfing 합성어로 AI 검색을 서핑을 하다는 컨셉으로 네이밍입니다.
심볼을 파도로 단순화해 작업을 했으며 청량한 블루 컬러로 파도의 느낌을 더했으며 통일성과 아이덴티티를 느낄 수 있도록 했습니다.
