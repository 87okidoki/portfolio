반려의고수

강아지와 함께하는 모든 순간 반려의고수

Overview

반려의고수는 모든 반려견과 반려견주가 더 나은 삶을 살아갈 수 있도록 도와주는 서비스입니다.
우리 댕댕이에게 필요한 정보 이젠 힘들게 찾지 않아도 반려의고수에서 상황별 맞춤 추천을 해드려요.

참여 : 웹 퍼블리셔

여러 환경의 사용자에게 동일한 사용자 경험을 제공하도록 반응형과 크로스 브라우징 제공하여 유연하게 대응 할 수 있도록 작업을 했습니다.
또한 검색 친화적인 웹사이트 구현을 위해 meta 요소를 이용한 정확한 문서 정보의 제공과 적정한 제목과 사용 의미에 맞는 마크업 작업을 해 검색의 효율성을 높여 주었습니다.
손쉬운 유지 보수를 위하여 구조와 표현을 분리하여 작업을 했으며 공통 컴포넌트를 따로 만들어 재활용이 가능해 소스 경량화가 되도록 했습니다.
